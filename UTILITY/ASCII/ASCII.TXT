                                    ASCII 
                                   
                         zobrazen tabulky znak ASCII

Pomoc  programu  ASCII  lze  zobrazovat tabulky systmovch font displeje. Po
sputn  programu se zobraz nejdve zkladn tabulka znak s kdem 0 a 127,
po  stisku klvesy (jin ne Esc) se zobraz druh polovina tabulky systmovch
font  s  kdy  128 a 255. Fonty se zobraz v aktulnm videomdu. Program lze
vyut  k  prohlen  font  displeje, uren vak je pedevm k tisku tabulek
font  na  tiskrnu.  Jako  pklad pouit je uveden povelov soubor ASCI.BAT,
kter  umouje  ve  spoluprci s ovladaem nrodnch font EGACS.COM zobrazit,
resp. vytisknout tabulky font displeje v kdu IBM, Kamenickch a Latin2. To se
realizuje  v  uvedenm  povelovm  souboru tak, e se nejdve nastav grafick
videomd  displeje  16  (640x350/16  barev).  Potom  se  nastav  ovlada font
displeje  EGACS.COM  na  pslun  kd  font a programem ASCII.COM se zobraz
tabulky  znak  v  tomto  kdu  font.  Zobrazen  tabulky  lze (za pedpokladu
instalace  grafickho ovladae tisku obrazovky GRAPHICS) vytisknout na tiskrnu
stiskem  klvesy  Print  Screen  nebo  sejmout rezidentnm programem grafickho
editoru.

-------------------------------------------------------------------------------
           Pklad - zobrazen a tisk tabulek font v rznch kdech
-------------------------------------------------------------------------------
@echo off
cls
egacs /k > nul:
vmod 3
echo Tento povelov soubor vm umon prohlet fonty instalovan pomoc
echo programu EGACS.COM. Po zobrazen font na displeji mete obsah displeje
echo vytisknout na pipojen tiskrn stiskem klvesy PrintScreen.
echo.
echo K tomu je zapoteb ptomnost nsledujcch program:
echo.
echo GRAPHICS.COM - systmov program pro tisk obrazovky v grafickm reimu
echo EGACS   .COM - instaltor nrodnch font displeje
echo VMOD    .COM - nastaven videomdu displeje
echo ASCII   .COM - zobrazen tabulek font displeje
echo WAIT    .COM - prodleva
echo.
echo Pokud vm nkter z program chyb, perute dal innost stiskem
echo klvesy Ctrl-Break.
echo.
echo.
wait
graphics graphics
vmod 16
egacs /0
echo.
echo Zobraz se standardn kd IBM.
echo.
echo.
wait
ascii
egacs /k
echo.
echo Zobraz se kd brat Kamenickch.
echo.
echo.
wait
ascii
egacs /l
echo.
echo Zobraz se kd Latin 2.
echo.
echo.
wait
ascii
egacs /k > nul:
