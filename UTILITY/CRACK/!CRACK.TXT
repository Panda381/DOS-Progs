  Program !CRACK je prostredek ke snadne modifikaci souboru a programu
  podle modifikacni tabulky. Tato tabulka je uchovana uvnitr programu.
  Je ji mozne definovat pomoci dale uvedenych parametru. Po kazde zmene
  tabulky je ulozen na disk novy soubor !CRACK.COM. Po provedeni vsech
  zmen lze program !CRACK.COM zkomprimovat a ochranit atributem R/O.

     /L ............... vypis aktualniho nastaveni programu
     /S data .......... vyhledani dat v modifikovanem souboru
     /H offset ........ zobrazeni 128 bajtu ze zadaneho offsetu
     /T soubor ........ soubor s textem hlaseni pri spusteni
     /F soubor ........ specifikace modifikovaneho souboru (programu)
     /V offset data ... provede se overeni dat na zadanem offsetu
     /W offset data ... provede se zapis dat na zadany offset

Offsety a data jsou v HEX kodu, jako data lze zadat i vice bajtu. Pri nezadani
zadnych dat se definice zrusi. Po spusteni programu bez parametru se zobrazi
hlaseni - parametr /T. Je-li stisknuta jina klavesa nez ESC nebo Ctrl-Break 
otevre se soubor (parametr /F), overi se data podle parametru /V a pri shode
se zapisou data podle parametru /W.
