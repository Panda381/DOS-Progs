Granularity

Filtr "Granularity" odstupòuje schodovitì úrovnì barevných kanálù RGB obrazu podle vztahu Yout = floor(Yin*Steps)/Steps. Alfa složka obrazu (prùhlednost) není filtrem ovlivnìna. Signál není ohranièen.

  • Source - obrazový vstup. Vstup není ohranièen.

  • Steps - poèet krokù. Úroveò signálu 0.01..1 odpovídá 1..100 krokùm, což odpovídá uživatelsky zadané hodnotì 1..100. Hodnota 0 není definována. Vstup není ohranièen.
