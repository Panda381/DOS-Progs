DivInv

Filtr "DivInv" vydìlí úrovnì barevných kanálù RGB druhého vstupu prvním vstupem podle vztahu Yout = Y2in / Y1in. Alfa složka obrazu (prùhlednost) zùstává zachována ze vstupu Dividend. Signál není ohranièen.

  • Source - první obrazový vstup. Vstup vydìlí hodnotu druhého vstupu. Vstup není ohranièen.

  • Divisor - druhý obrazový vstup. Vstup je vydìlen hodnotou prvního vstupu. Vstup není ohranièen.
