Kaleidoscope

Filtr "Kaleidoscope" deformuje obraz jako kaleidoskop.

  • Source - obrazový vstup. Vstup není ohranièen.

  • Mode - volba módu kaleidoskopu. Platné hodnoty jsou 0 až 0.6, s krokem 0.1.
